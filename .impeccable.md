# Nawa Studio — Design Context

> Generated by `/teach-impeccable`. Update this file when brand direction evolves.

## Design Context

### Users

**Primary audience** — B2B decision-makers at French SMEs and mid-market companies:
- DRH / Responsables RH looking to automate candidate sourcing & screening
- CEOs / COOs / Ops Directors wanting to deploy AI without a tech team
- Context: discovery phase (first visit), evaluating credibility and fit before contacting
- Job to be done: "Understand what Nawa Studio does, believe it's real and trustworthy, take action"

**Secondary audience** — Tech-savvy early adopters curious about AI automation.

### Brand Personality

**Three words**: Premium · Trustworthy · Forward-thinking

The brand should feel like a boutique engineering firm — not a startup hype machine. Confidence without arrogance. Precision without coldness. Like working with a small elite team that makes complex things simple.

**Voice**: Direct, concrete, reassuring. French-language. No buzzword inflation. Results over promises.

**Emotional goal**: Visitors should feel "these people know what they're doing — I want to work with them."

### Aesthetic Direction

**Visual tone**: Refined minimalism — inspired by Linear, Vercel, Anthropic. Clean whitespace, sharp typography, purposeful color.

**Theme**: Light (white base `#FFFFFF`, surface `#F8F6FF`)

**Color system** (actual tokens in `globals.css`):
- `--bg`: `#FFFFFF`
- `--surface`: `#F8F6FF`
- `--border`: `#E2DAF6`
- `--text-primary`: `#111827`
- `--text-secondary`: `#4B5563`
- `--text-muted`: `#9CA3AF`
- `--accent-blue`: `#7C63C8` (primary — purple-indigo, not electric blue)
- `--accent-blue-light`: `#EEE9FB`
- `--accent-violet`: `#B8AEDE`
- `--accent-violet-light`: `#DDD8F5`

**Typography**:
- Headings: Space Grotesk, weight 700–800, tight tracking (`-0.03em`)
- Body: Inter, weight 400, line-height 1.7
- CSS variables: `--font-heading`, `--font-body`

**References**: Linear.app, Vercel.com, Anthropic.com
**Anti-references**: SaaS with dark glassmorphism + neon, generic Bootstrap sites, "AI startup" templates with cyan/teal palettes

### Design Principles

1. **Precision over decoration** — Every visual element must earn its place. No gradient text unless it carries meaning. No cards inside cards.

2. **Typography does the heavy lifting** — Headlines at `clamp(44–82px)` with tight tracking create impact without decoration. Let size and weight speak.

3. **Whitespace is content** — Generous breathing room signals premium. Tight spacing signals cheap.

4. **One clear action per moment** — Each section has one job. The hero converts. The catalogue informs. No competing CTAs at the same visual weight.

5. **Motion reveals, never distracts** — Framer Motion `fadeUp` with 0.65s + `[0.22, 1, 0.36, 1]` easing. `viewport={{ once: true }}` on all scroll sections. Never bounce, never elastic.

### Component Conventions

- Buttons: `border-radius: 14px`, purple fill (`#7C63C8`), `box-shadow` lift on hover, `translateY(-2px)` on hover
- Badge: uppercase micro-caps, `border: 1px solid rgba(124,99,200,0.2)`, pulsing dot
- Sections: `py-24` desktop, `py-16` mobile
- Cards: subtle border `#E2DAF6`, `#F8F6FF` background, no inner cards

### Stack Constraints

- Next.js 14 App Router — Server Components by default, `"use client"` only when needed
- Tailwind CSS v4 — no external CSS libraries
- TypeScript strict — no `any`
- Framer Motion via `LazyMotion` + `domAnimation`
- No Spline currently (removed from Hero)
- Deployed on Vercel
